Présentation

jisql est un éditeur Transact-SQL facile à utiliser, qui vous aide à :
  • Composer des instructions de commande T-SQL avec une interface utilisateur graphique
  • Exécuter des requêtes T-SQL
  • Visualiser les résultats des requêtes au format texte ou table
  • Enregistrer les données entrées ou les résultats dans des fichiers de formats courants, faciles à importer dans d'autres applications (par exemple, Microsoft Excel ou bcp)
Entièrement écrit en Java, jisql tire parti des JFC (Java Foundation Classes) de Swing.

jisql est intuitif et suppose que vous êtes déjà familiarisé avec Transact-SQL et les serveurs de bases de données Sybase.

La fenêtre de connexion contient des boutons et des listes de sélection, la fenêtre de saisie, la fenêtre de résultats, la fenêtre d'état et le menu amovible. Il est possible de redimensionner la hauteur de chaque fenêtre en faisant glisser à l'aide de la souris la ligne horizontale qui les sépare.

jisql supporte des jeux de caractères codés sur plusieurs octets pour afficher les données dans différentes polices. Vous devez vérifier que l'environnement de votre ordinateur est correctement défini. Par exemple, sous UNIX, vous devez faire particulièrement attention aux variables d'environnement LC_CTYPE et LANG. Vérifiez également que les polices que vous souhaitez utiliser sont bien installées sur votre machine.

jisql supporte Input Method Framework, qui permet de taper un texte dans la fenêtre de saisie en chinois, en japonais ou en coréen (langues qui utilisent des milliers de caractères différents) sur un clavier à caractères latins. Pour plus d'informations, reportez-vous à la page Web Input Method Framework Overview de Sun Microsystem.
