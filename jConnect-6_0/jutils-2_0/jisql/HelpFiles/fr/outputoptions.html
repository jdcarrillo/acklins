Options de la fenêtre de résultats

Ces options permettent de définir facilement les paramètres de la fenêtre de résultats. Elles sont spécifiques de la connexion. Cliquez sur Appliquer pour enregistrer vos modifications.

OPTION DESCRIPTION
Présenter les jeux de résultats sous forme de (Texte/Table)

Indique la présentation des futurs jeux de résultats dans la fenêtre. Cette option ne s'applique pas au jeu de résultats courant.

L'option par défaut est "Table".

Voir aussi : Résultats au format table ou texte.

Méthode utilisée dans l'objet DB pour afficher les données Cette option désigne la méthode utilisée pour imprimer les valeurs de données enregistrées dans un objet Java, lui-même stocké dans la base de données. La valeur par défaut est "toString", mais vous pouvez la remplacer par un nom de méthode défini par l'utilisateur.
Extraire automatiquement les données de l'objet Java DB Cochez cette case pour extraire automatiquement les données des objets Java stockés dans la base de données.
Délimiteur de champ des fichiers de résultats (table seulement) Définit le caractère à utiliser pour délimiter les données de champ lorsqu'un jeu de résultats est enregistré dans un fichier. Cette option n'est disponible que si le jeu de résultats est présenté sous forme de table. Les jeux de résultats affichés sous forme de texte ont exactement le même aspect que dans le fichier de résultats.
