Options diverses

Cliquez sur Appliquer pour enregistrer vos modifications.
 
OPTION DESCRIPTION
Des avertissements SQL devraient être affichés Choisissez le mode d'affichage des exceptions SQL et des avertissements :
  • Dans une fenêtre texte distincte 
  • Dans une boîte de dialogue
  • Dans la fenêtre d'état (sous la fenêtre de résultats)
  • Ne pas afficher
L'option par défaut est "Ne pas afficher".
Nbre maximal de batchs dans l'historique Indiquez le nombre d'instructions que jisql doit stocker dans le buffer de l'historique. Une instruction peut se composer d'une ou plusieurs lignes dans un batch. La valeur par défaut est 50. Pour un nombre illimité, entrez 0 (zéro).
Définir la police Utilisez la liste déroulante pour changer de police et/ou de taille de caractères dans la fenêtre de la session active. Les polices des textes de libellés et de bordures sont modifiées mais la taille reste la même. Si un style de police et une taille ne peuvent pas être associés, les caractères ne s'affichent pas du tout.
