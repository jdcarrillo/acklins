Menu Liste

OPTION
DESCRIPTION
Afficher les tables utilisateur

Cette option affiche la liste des tables utilisateur de la base de données courante. Cliquez deux fois sur le nom d'une table pour ouvrir ses colonnes, y compris des éléments de la définition de la table tels que le nom des colonnes, le type des données et leur longueur.

Si vous souhaitez créer une instruction select pour une table ou une colonne, cliquez sur le nom de celle-ci puis sur Select. Pour sélectionner plusieurs noms non contigus, maintenez la touche Contrôle enfoncée et cliquez sur chaque nom à inclure dans la sélection. Pour sélectionner plusieurs noms contigus, maintenez la touche Majuscules enfoncée et cliquez sur les premier et dernier noms de la sélection. jisql insère une instruction select pour les tables ou les colonnes choisies dans la fenêtre de saisie.
Afficher les tables système

Cette option affiche les tables système de la base de données courante. Cliquez deux fois sur le nom d'une table pour afficher ses colonnes, y compris des éléments de la définition de la table tels que le nom des colonnes, le type des données et leur longueur.

Si vous souhaitez créer une instruction select pour une table ou une colonne, cliquez sur le nom de celle-ci puis sur Select. Pour sélectionner plusieurs noms non contigus, maintenez la touche Contrôle enfoncée et cliquez sur chaque nom à inclure dans la sélection. Pour sélectionner plusieurs noms contigus, maintenez la touche Majuscules enfoncée et cliquez sur les premier et dernier noms de la sélection. jisql insère une instruction select pour les tables ou les colonnes choisies dans la fenêtre de saisie.
Afficher les procédures stockées

Cette option affiche la liste des procédures stockées dans la base de données courante. Cliquez sur le nom d'une procédure puis sur Afficher texte pour afficher le texte de la procédure sélectionnée. Il est possible d'enregistrer ce texte dans un fichier et de le modifier.

Si vous voulez utiliser une partie du texte dans une requête, sélectionnez celui-ci, appuyez sur Ctrl+c, placez votre curseur dans la fenêtre de saisie et appuyez sur Ctrl+v. jisql insère le texte copié dans la fenêtre. Cliquez sur Soumettre pour exécuter l'instruction. Toutefois, cette opération ne fonctionne pas avec tous les gestionnaires de fenêtres.
