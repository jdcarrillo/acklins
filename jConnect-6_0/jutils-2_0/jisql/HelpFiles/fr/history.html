Fenêtre d'historique

Cliquez sur le bouton Historique pour ouvrir la fenêtre d'historique qui affiche tous les batchs précédemment exécutés (comprenant une ou plusieurs instructions) pour la session et la connexion en cours. Le nombre de batchs par défaut enregistré par jisql est de 50 ; 0 (zéro) indique que ce nombre est illimité. Pour changer cette valeur, utilisez l'onglet Options diverses de la fenêtre des options.

Vous pouvez :

  • Modifier le fichier, par exemple lui ajouter des commentaires.

  • Cliquer sur Enregistrer dans un fichier pour enregistrer le contenu de la fenêtre d'historique dans un fichier texte.

  • Sélectionner le texte à réutiliser et cliquer sur Soumettre pour réafficher l'instruction ou le batch dans la fenêtre de saisie et l'exécuter à nouveau.

  • Cliquer sur l'icône Haut pour sélectionner le batch précédent dans la fenêtre d'historique et l'afficher dans la fenêtre de saisie où vous pouvez modifier la commande ou cliquer sur Soumettre pour une réexécution.

  • Cliquer sur l'icône Bas pour sélectionner le batch suivant dans la fenêtre d'historique et l'afficher dans la fenêtre de saisie où vous pouvez modifier la commande ou cliquer sur Soumettre pour une réexécution.
