Messages d'erreur

Une fenêtre contenant un message d'erreur s'affiche chaque fois qu'une exception SQL ou un avertissement est généré par la base de données, jConnect ou la machine virtuelle Java. Cette fenêtre contient un bouton "Afficher stacktrace" qui ouvre une nouvelle fenêtre permettant de visualiser la stacktrace. Le cas échéant, celle-ci peut être enregistrée dans un fichier afin de faciliter la tâche du Support Technique.
