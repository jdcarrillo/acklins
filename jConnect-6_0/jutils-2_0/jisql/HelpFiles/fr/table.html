Résultats au format table ou texte

Les jeux de résultats peuvent être présentés dans un composant Swing appelé JTable ou dans une zone de texte normale. Vous pouvez choisir le format souhaité dans l'onglet Options de la fenêtre de résultats du menu Options.

Les avantages de JTable sont les suivants :

  • Aspect plus convivial
  • Résultat plus convivial des données dans un fichier, y compris la préparation des données de type bcp
  • Possibilité de réorganiser les colonnes en faisant glisser les en-têtes
  • Manipulation aisée des colonnes de type image (voir Colonnes de type image)
  • Manipulation aisée des colonnes de type objet Java
  • Affichage séparé de chaque jeu de résultats

Les avantages des zones de texte sont les suivants :

  • Affichage des résultats plus rapide
