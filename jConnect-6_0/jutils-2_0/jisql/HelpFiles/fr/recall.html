Rappel de requêtes précédentes

Pour rappeler et exécuter une requête précédemment enregistrée, cliquez sur l'icône Historique dans la barre des boutons pour ouvrir la fenêtre d'historique.

Cette fenêtre affiche les batchs précédemment exécutés, qui sont stockés dans le buffer de l'historique pour la connexion courante. Pour changer le nombre de scripts enregistrés dans le buffer de l'historique, cliquez sur l'onglet Options diverses dans la fenêtre des options.

Vous pouvez :

  • Sélectionner le texte à réutiliser et cliquer sur Soumettre pour réafficher l'instruction ou le batch dans la fenêtre de saisie et recommencer l'exécution.

  • Cliquer sur l'icône Haut pour sélectionner le batch précédent dans la fenêtre d'historique et l'afficher dans la fenêtre de saisie où vous pouvez modifier la commande, ou cliquer sur Soumettre pour une nouvelle exécution.

  • Cliquez sur l'icône Bas pour sélectionner le batch suivant dans la fenêtre d'historique et l'afficher dans la fenêtre de saisie où vous pouvez modifier la commande, ou cliquer sur Soumettre pour une nouvelle exécution.
