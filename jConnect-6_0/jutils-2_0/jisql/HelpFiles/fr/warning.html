Erreurs et avertissements SQL

Les erreurs et avertissements SQL contiennent généralement des messages d'information provenant de la base de données. Par défaut, chaque avertissement s'affiche dans une fenêtre distincte. Toutefois, vous pouvez choisir de les afficher dans la fenêtre d'état afin de les imprimer ou de désactiver leur affichage.
