Menu Edition

OPTION
DESCRIPTION
Couper

Permet de couper des données d'une autre application Java et de les coller dans jisql. Vous pouvez aussi couper des données de jisql pour les coller dans une autre application Java, à condition que celle-ci supporte la fonction Couper.
Copier Permet de copier des données d'une autre application Java et de les coller dans jisql. Vous pouvez aussi copier des données de jisql pour les coller dans une autre application Java, à condition que celle-ci supporte la fonction Coller.
Coller Permet de coller des données dans la fenêtre de saisie.
REMARQUE Sur les plates-formes Microsoft, les fonctions Couper, Copier et Coller (y compris les raccourcis Ctrl-c et Ctrl-v) sont prises en charge entre jisql et des applications natives (telles que XTerm ou MS Word). Sur les plates-formes UNIX, elles ne semblent être reconnues que dans les gestionnaires Motif tels que CDE et mvm.
