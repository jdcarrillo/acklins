Soumettre
Cliquez sur Soumettre pour exécuter la ou les instructions affichées dans la fenêtre de saisie.
Base de données

Cette zone de liste déroulante permet d'afficher toutes les bases de données associées au serveur nommé dans la barre d'outils. Pour choisir rapidement une nouvelle base de données dans le cadre d'une session, cliquez sur son nom dans la zone de liste déroulante. Lorsque vous sélectionnez une base de données différente, une ligne "use [database]" est ajoutée au buffer de l'historique.

Vous pouvez aussi choisir une autre base de données en exécutant une commande use <database> depuis la fenêtre de saisie. Lorsque vous créez, modifiez ou supprimez une base de données dans la fenêtre de saisie, la liste déroulante dans la barre d'outils n'est pas mise à jour.

Cliquez sur l'icône Historique pour ouvrir une fenêtre d'historique, qui affiche tous les batchs précédents (constitués d'une ou plusieurs instructions) pour la connexion en cours. Pour spécifier le maximum de batchs qu'il est possible de stocker dans le buffer de l'historique, utilisez l'onglet Options diverses de la fenêtre des options.

Sélectionnez le texte à réutiliser et cliquez sur Soumettre pour réafficher l'instruction ou le batch dans la fenêtre de saisie et recommencer l'exécution.
Cliquez sur l'icône Haut pour sélectionner le batch précédent dans la fenêtre d'historique et l'afficher dans la fenêtre de saisie où vous pouvez éditer la commande, ou cliquez sur Soumettre pour une nouvelle exécution.
Cliquez sur l'icône Bas pour sélectionner le batch suivant dans la fenêtre d'historique et l'afficher dans la fenêtre de saisie où vous pouvez éditer la commande, ou cliquez sur Soumettre pour une nouvelle exécution.
