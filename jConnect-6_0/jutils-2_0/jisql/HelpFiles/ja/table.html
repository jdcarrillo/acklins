テーブル形式とテキスト形式の出力

ResultSet は、JTable と呼ばれる Swing コンポーネントとして、または通常のテキストとして提示されます。 どちらの形式を使って表示するかは、［オプション］メニューの［出力ウィンドウ・オプション］で設定できます。

JTable を使用する利点は、次のとおりです。

  • 出力表示が整っている。
  • ファイルへの書き出し時のフォーマットが整っている（bcp で準備されたデータに類似）。
  • テーブル・ヘッダをドラッグ・アンド・ドロップしてカラムの並べ替えができる。
  • イメージ・カラムを簡単に処理できる（詳細はイメージ・カラムを参照）。
  • Java オブジェクト・カラムを簡単に処理できる。
  • 複数の ResultSet がある場合は、それぞれ個別のタブに表示される。

テキストを使用する利点は、次のとおりです。

  • 出力速度が速い。
