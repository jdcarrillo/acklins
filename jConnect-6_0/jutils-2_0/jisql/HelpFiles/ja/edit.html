［編集］メニュー

オプション
説明
［切り取り］

他の Java アプリケーションからデータを切り取って、jisql へ貼り付けることができます。 同様に、jisql の内容を切り取って、貼り付け機能をサポートしている他のアプリケーションに貼り付けることもできます。
［コピー］ 他の Java アプリケーションからデータをコピーして、jisql へ貼り付けることができます。 同様に、jisql の内容をコピーして、貼り付け機能をサポートしている他のアプリケーションに貼り付けることもできます。
［貼り付け］ ［入力］ウィンドウ内でデータの貼り付けを行うことができます。
注意 jisql と他のアプリケーション（XTerm や MS Word など）間での切り取り、コピー、および貼り付け機能（Ctrl -c や Ctrl-v などのショートカットを含む）は、Microsoft プラットフォームで利用できます。 UNIX では、これらの機能は CDE や mvm などの Motif ベースのウィンドウ・マネージャでのみ利用できます。
