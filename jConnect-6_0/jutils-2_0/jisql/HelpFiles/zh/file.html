“文件”菜单

选项
说明
读取批处理文件

允许执行预先准备好的批处理文件。根据在输入窗口选项选项卡中定义的状态，可将文件读取到输入窗口中（在此窗口中可编辑该文件），然后用“执行”按钮或者用“查询”菜单的执行选项执行该文件。可在输入窗口选项选项卡中设置命令行分隔符。

要在后台执行批处理，只需在输入窗口选项选项卡中选中相应的选项即可。
保存输入窗口的内容到文件中 将输入窗口中的当前内容保存到文件中。稍后，可以再次读取和执行该文件。
保存输出缓冲内容到文件中

每个 ResultSet 都存储在一个输出缓冲区中。此选项可将输出缓冲区中的内容保存到文件中。这是保存存储在数据库中的数据的一种简单方法。如果数据显示为一个表组件，则用输出窗口选项选项卡可以定义字段分隔符。这使您可以准备一个输出文件，该文件将在以后与 bcp 一起使用，以便将数据读回数据库的表中。行分隔符总是换行符。

如果数据显示为文本，则存储的内容就是输出窗口中的显示内容。
保存历史缓冲区内容到文件中 您执行的每一条命令都存储在历史缓冲区中。单击历史图标后将显示该缓冲区的内容。为了日志记录或为了帮助找到失败的原因，可以编辑这些内容（如增加注释），并将历史缓冲区内容保存到文件中。
打印 ResultSets ASCII 风格

将当前的 ResultSet 发送给打印机打印。数据显示为 ASCII 文本。

注释：  如果在“输出窗口”中改变数据（例如，调整列的大小或重新排序列），则这些改变不会表现在打印输出中。
以 JTable 风格打印 ResultSet
(仅适用于 jdk 1.2 )

将当前的 ResultSet 发送给打印机打印。数据显示为 JTable 格式。

注释：  如果在“输出窗口”中改变数据（例如，调整列的大小或重新排序列），则这些改变不会表现在打印输出中。
退出 关闭所有会话窗口（这也将关闭相关的数据库连接），并结束应用程序进程。
