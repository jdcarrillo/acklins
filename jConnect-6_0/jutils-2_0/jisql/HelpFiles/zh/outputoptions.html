输出窗口选项

这些选项提供了一种设置“输出窗口”参数的简便方法。这些选项与特定连接有关。单击应用保存您所作的改变。

选项 说明
将 ResultSet 显示为（文本/表）

指定如何在“输出窗口”中显示今后的 ResultSet。当前的 ResultSet 不受影响。

缺省为“表”。

参见：表与文本输出。
DB Object 中显示数所使用的方法 该方法用于输出存储在 Java 对象（此对象存储在数据库中）中的数据值。缺省为“toString”，但可以将其改为用户自定义的方法名。
自动从 DB Java Object 中抽取数据 选中此复选框可以自动从数据库中存储的 Java 对象中抽取数据。
输出文件中使用的字段分隔符（仅适用于表） 定义将 ResultSet 写入文件时字段数据间的用作分隔符的字符。这只在 ResultSet 显示在表中时才有效。在输出文件中以文本模式显示的 ResultSet 与在“输出窗口”中完全相同。
