“编辑”菜单

选项
说明
剪切

可从其它 Java 应用程序剪切并粘贴内容到 jisql。如果 Java 应用程序支持“剪切”功能，也可以从 jisql 剪切内容到该应用程序中。
复制 可从其它 Java 应用程序复制并粘贴内容到 jisql。如果 Java 应用程序支持“复制”功能，也可以从 jisql 复制内容到该应用程序。
粘贴 可在“输入”窗口中粘贴内容。
注释 jisql 和本地应用程序 （如 XTerm 或 MS Word）之间的剪切、复制和粘贴操作（包括快捷键 Ctrl -c 和 Ctrl -v）在 Microsoft 平台上有效。在 UNIX 平台上，仅在基于 Motif 的窗口管理器（如 CDE 和 mvm）中才可能进行这些操作。
