表与文本输出

ResultSet 可以在一个称为 Jtable 的 Swing 组件中或在常规文本区中显示。可以使用“选项”菜单的输出窗口选项面板选择两种格式之一。

使用 Jtable 的优点是：

  • 外观更漂亮
  • 数据输出到文件更顺畅，包括类似于 bcp 的数据准备工作
  • 可以拖放表标题对表中各列进行重新排序
  • 图像列的处理方便（参见图像列）
  • Java 对象列的处理方便
  • 当有多个 ResultSet 时，每个 ResultSet 均将显示在一个单独的选项卡上

使用文本区的优点是：

  • 输出更快
