概述

jisql 是一种易用的 Transact-SQL (T-SQL) 编辑器，可帮助您：
  • 使用图形用户界面编写 T-SQL 命令语句
  • 执行 T-SQL 查询
  • 以文本方式或在表中查看查询结果
  • 使用可以导入其它应用程序的通用格式保存输入数据和结果数据（例如，Microsoft Excel 或 bcp）
jisql 是使用 Java 编写的，利用了 Java 基础类 (JFC) 的优点，JFC 是 Swing 的一部分。

jisql 设计得非常直观，并且假定您拥有 Transact-SQL 和 Sybase 数据库服务器的使用知识。

连接窗口包括按钮和选择列表、输入窗口、输出窗口、状态窗口和可拆分的菜单。通过拖动水平分割线来分别调整各窗口的高度。

jisql 支持多字节字符集，可以使用不同的字体显示数据。必须确认机器的环境设置是否正确；例如，UNIX 需要特别注意 LC_CTYPE 和 LANG 环境变量。此外，还需要确认在机器上是否确实安装了要使用的字体。

jisql 支持输入法框架 (Input Method Framework)，使您可以使用罗马字符的键盘在“输入窗口”中键入中文、日文或韩文以及多种使用不同字符的语言的文字。 详细信息请参见 Sun Microsystem 的输入法框架概述。
