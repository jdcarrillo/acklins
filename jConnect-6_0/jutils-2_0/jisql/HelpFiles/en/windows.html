Windows

jisql may have zero, one, or more connections open at any given time, each having its own session window. Each new session window has:

  • an Input Window where you enter your query statements
  • an Output Window that displays the results of an executed queries
  • a Status Window located below the Output Window, that displays query execution error and informational messages, and
  • the tool bar that contains the Go button to execute queries, a select list of the databases that reside on the server to which you have a connection, and the History icons that help you display previously executed queries.
