Edit Menu

OPTION
DESCRIPTION
Cut

Allows you to Cut from other Java applications and Paste into jisql. You may also Cut from jisql into another Java application when Cut functionality is supported in that application.
Copy Allows you to Copy from other Java applications and Paste into jisql. You may also Copy from jisql into another Java application when Paste functionality is supported in that application.
Paste Allows you to paste work inside the Input Window.
NOTE Cut, Copy, and Paste between jisql and native applications (like XTerm or MS Word) works on Microsoft platforms (including the shortcuts Ctrl -c and -v). On UNIX platforms it may work only on Motif-based Window-Managers like CDE and mvm.
