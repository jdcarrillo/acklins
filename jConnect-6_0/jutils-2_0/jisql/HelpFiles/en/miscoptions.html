Miscellaneous Options

Click Apply to save your changes.
 
OPTION DESCRIPTION
SQL Warnings should be presented Specify how you want SQL Exceptions or warnings to display:
  • In a separate text window 
  • In a PopUp Window
  • In the Status Window (below the Output Window)
  • Not at all
The default is "not at all."
Max statements in history Specify the number of statements jisql should store in the history buffer. A statement can consist of one line, or several lines in a batch. The default is 50. Enter zero (0) for unlimited.
Set Font Use the drop-down list box to change the font and/or size used in the active session window. Fonts for label text and border text will change,but their font sizes stay the same. If a font style and size combination does not work together, the characters do not display at all.
