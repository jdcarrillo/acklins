Options Menu

The option on this menu opens the jisql Options Window, which allows you to set new values for the Output Window, the Input Window, the database, database diagnostics, and miscellaneous.

These values are connection-specific, but may not show the actual current value for a connection. The purpose of these options is to allow you a convenient way to set option values.

Applying new option values affects only the currently "active" session window. If there is more than one open session window, the active window is the one in front of the other session windows with a title bar in a different or more intense color.
