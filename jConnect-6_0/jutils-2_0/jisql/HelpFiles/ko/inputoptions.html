입력 창 옵션

  • 배치에 사용할 명령 구분자
    명령 구분자는 질의문의 끝을 표시하고 질의문을 데이터베이스 서버에 보내 실행되도록 합니다. 명령 구분자의 기본값은 Sybase의 Transact-SQL 구분자인 'go'입니다.

    'go' 이외의 다른 명령 구분자를 사용하려면 다른 키워드를 입력합니다.

    참고:     공백은 삭제됩니다. 즉, ' go '와 'go'는 동일하게 처리됩니다.

  • 파일에서 읽은 배치를 입력 창에 표시
    기본으로 설정되어 있는 이 옵션을 사용하면 불러온 배치 파일이 입력 창에 표시되므로, 바로 편집하여 실행 단추나 질의 메뉴의 실행 옵션을 눌러 실행할 수 있습니다. 이 옵션을 선택하지 않으면 배치가 백그라운드에서 실행됩니다. 결과 및 상태 메시지는 옵션 선택 여부와 상관없이 나타납니다.
  • 첫번째 예외 발생 시 배치 취소?

    이 옵션을 선택하면 큰 배치를 실행하는 도중 SQL 예외가 발생할 경우 jisql이 배치 실행을 취소합니다. 이 옵션은 기본으로 설정되어 있습니다.
