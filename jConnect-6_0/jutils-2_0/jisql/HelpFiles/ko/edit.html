잘라내기

다른 Java 응용 프로그램에서 잘라내어 jisql에 붙여 넣을 수 있습니다. 다른 Java 응용 프로그램에서 잘라내기 기능을 지원할 경우 jisql에서 잘라내어 붙여 넣을 수도 있습니다.
복사 다른 Java 응용 프로그램에서 복사하여 jisql에 붙여 넣을 수 있습니다. 다른 Java 응용 프로그램에서 복사 기능을 지원할 경우 jisql에서 복사하여 붙여 넣을 수도 있습니다.
붙여넣기 입력 창에서 붙여넣기 작업을 할 수 있습니다.
참고 jisql과 원시 응용 프로그램(XTerm 또는 MS Word 등) 간의 잘라내기, 복사, 붙여넣기 명령(바로 가는 키 Ctrl+C 와 Ctrl+V 포함)은 Microsoft 플랫폼에서 사용할 수 있습니다. UNIX 플랫폼에서는 CDE나 mvm 같은 Motif 기반의 창 관리자에서만 사용할 수 있습니다.
