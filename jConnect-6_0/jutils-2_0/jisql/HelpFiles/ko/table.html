표 형식과 텍스트 형식의 출력

결과 집합은 JTable이라고 하는 Swing 구성 요소나 일반 텍스트 영역에 표시됩니다. 옵션 메뉴의 출력 창 옵션 영역에서 표시 형식을 선택할 수 있습니다.

JTable을 사용하면 다음과 같은 이점이 있습니다.

  • 세련된 외관
  • bcp와 유사한 데이터 처리 등 뛰어난 데이터 파일 출력
  • 표 머리글을 끌어 놓아 열 재정렬 가능
  • 편리한 이미지 열 처리(이미지 열 참고)
  • 편리한 Java 객체 열 처리
  • 여러 개의 결과 집합이 있을 경우 별개의 탭에 각 결과 집합 표시

텍스트 영역을 사용하면 다음과 같은 이점이 있습니다.
