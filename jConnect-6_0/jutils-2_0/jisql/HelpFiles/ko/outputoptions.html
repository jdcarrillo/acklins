출력 창 옵션

이러한 옵션을 사용하여 출력 창 매개변수를 쉽게 설정할 수 있습니다. 이 옵션은 각 연결마다 다르게 적용할 수 있습니다. 변경 사항을 저장하려면 적용을 누릅니다.

옵션
설명
결과 집합 표시 형식(텍스트/표)

이후의 결과 집합을 출력 창에 표시할 방법을 지정합니다. 현재의 결과 집합에는 적용되지 않습니다.

기본값은 "표"입니다.

참조: 펴 형식 및 텍스트 형식 출력.

데이터를 나타내기 위해 DB 객체에 사용되는 메소드 데이터베이스에 저장된 Java 객체의 데이터 값을 출력하는 데 사용된 메소드입니다. 기본값은 "toString"이지만 사용자 정의 메소드 이름으로 변경할 수 있습니다.
DB Java 객체에서 자동으로 데이터 추출 이 확인란을 선택하면 데이터베이스에 저장된 Java 객체에서 자동으로 데이터를 추출합니다.
출력 파일에 사용할 필드 구분자(표 형식만 해당) 결과 집합이 파일에 기록될 때 필드 데이터 간의 구분자로 사용할 문자를 정의합니다. 이 옵션은 결과 집합이 표 형식으로 표시되는 경우에만 적용됩니다. 텍스트 모드의 결과 집합은 출력 창에서 보이는 것과 동일하게 파일에 기록됩니다.
