기타 옵션

변경 사항을 저장하려면 적용을 누릅니다.
 
옵션
설명
SQL 경고 표시 SQL 예외나 경고의 표시 방식을 지정합니다.
  • 별도의 텍스트 창에 표시 
  • 팝업 창에 표시
  • 상태 창(출력 창 아래)에 표시
  • 표시 안함
기본 값은 "표시 안함"입니다.
히스토리의 최대 구문 수 jisql이 히스토리 버퍼에 저장할 구문의 수를 지정합니다. 구문은 한 행이나 여러 행의 배치로 구성될 수 있습니다. 기본값은 50입니다. 0은 무제한을 나타냅니다.
글꼴 설정 드롭다운 목록 상자를 사용하여 현재 세션 창에서 사용하는 글꼴이나 크기를 변경합니다. 레이블 텍스트와 테두리 텍스트는 글꼴은 변경할 수 있으나 글꼴 크기는 변경되지 않습니다. 글꼴 유형과 크기 조합이 적절하지 않으면 글자가 표시되지 않습니다.
