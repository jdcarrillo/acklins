질의 입력과 실행

배치 파일이나 스크립트는 작업을 수행하는 데 사용되는 Transact-SQL 구문의 모음입니다. 스크립트는 파일로 저장되며 대다수의 서버 응용 프로그램 프로젝트가 이들 스크립트를 기반으로 구축됩니다. 일반적으로 테이블, 인덱스, 내장 프로시저, 보고서, 배치 작업 등의 정의가 스크립트 파일에 저장됩니다.

서버에 연결한 다음 데이터베이스 드롭다운 목록을 누르고 작업할 데이터베이스의 이름을 선택합니다. 여러 가지 방법으로 질의를 입력할 수 있습니다. 질의를 실행하려면 도구 모음의 실행 단추를 누르거나 질의 메뉴에서 질의 실행을 선택합니다.

  • 입력 창에 질의를 직접 입력하고 실행을 누릅니다.
  • 목록 메뉴에서 옵션을 선택합니다.
    목록에 사용자 테이블이나 시스템 테이블이 표시될 경우 테이블이나 열 이름을 선택하고 선택을 누릅니다. 서로 떨어져 있는 여러 개의 이름을 선택하려면 Ctrl 키를 누른 채 원하는 이름들을 누릅니다. 서로 인접해 있는 여러 개의 이름을 선택하려면 Shift 키를 누른 채 선택할 이름 중 첫번째 이름과 마지막 이름을 누릅니다. jisql은 입력 창에서 선택한 테이블이나 열에 대해 select 구문을 삽입합니다.
    내장 프로시저나 트리거가 나열될 경우 표시할 항목의 이름을 선택하고 텍스트 보기를 누릅니다. 창이 열리면서 내장 프로시저나 트리거의 텍스트가 표시됩니다. 질의에 텍스트 일부를 사용하려면 사용할 텍스트를 선택하여 강조 표시하고 Ctrl+C를 누른 다음 입력 창에 커서를 놓고 Ctrl+V를 누릅니다. jisql은 창에 복사해 넣은 텍스트를 일부 창 관리자에 삽입합니다.
  • 입력 창에서 이전 질의를 선택하여 강조 표시하고 실행을 누르거나 질의 메뉴에서 질의 실행을 선택합니다.
  • 히스토리 창의 구문을 입력 창으로 전송합니다. (히스토리 창에 구문을 표시하려면 하나 이상의 구문이나 배치를 실행해야 합니다.)

    히스토리 아이콘 을 누릅니다. 히스토리 창에서 실행할 구문을 선택하여 강조 표시하고 연결 창의 단추 모음에서 실행을 누릅니다. Ctrl+C를 사용하여 히스토리 창에서 구문을 복사하고 Ctrl+V를 사용하여 복사한 구문을 입력 창에 붙여 넣을 수도 있습니다.

결과 집합이 출력 창에 표시됩니다.
