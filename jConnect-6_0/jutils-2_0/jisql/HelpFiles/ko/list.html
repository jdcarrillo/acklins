사용자 테이블 목록

현재 데이터베이스의 사용자 테이블 목록을 표시합니다. 테이블 이름을 두 번 누르면 열 이름, 데이터 형식, 데이터 형식의 길이와 같은 테이블 정의를 포함하는 테이블 열이 나타납니다.

특정 테이블이나 열을 선택하는 질의문을 작성하려면 테이블이나 열의 이름을 누른 다음 선택을 누릅니다. 연속되지 않은 이름을 여러 개 선택하려면 Ctrl 키를 누른 채 선택할 각 이름을 마우스 단추로 누릅니다. 연속으로 나열된 여러 개의 이름을 선택하려면 Shift 키를 누른 채 선택할 이름 그룹의 첫번째 이름과 마지막 이름을 마우스로 단추로 누릅니다. jisql은 선택한 테이블이나 열에 대한 선택 문을 입력 창에 삽입합니다.

내장 프로시저 목록

현재 선택한 데이터베이스의 내장 프로시저 목록을 나타냅니다. 내장 프로시저 이름을 누른 후 텍스트 보기를 누르면 선택한 프로시저의 텍스트가 나타납니다. 이 텍스트를 파일에 저장하여 편집할 수 있습니다.

텍스트의 일부를 질의에 사용하려면 해당 텍스트를 선택하고 Ctrl+C를 누른 후, 커서를 입력 창에 놓고 Ctrl+V를 누릅니다. jisql은 복사된 텍스트를 창에 삽입합니다. 질의문을 실행하려면 실행을 누릅니다. 그러나 일부 창 관리자에서는 이 작업이 허용되지 않습니다.
