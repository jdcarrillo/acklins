파일 메뉴

옵션
설명
배치 파일 읽기

배치 파일을 실행할 수 있습니다. 입력 창 옵션 탭에 정의된 상태에 따라 편집 가능한 입력 창으로 파일을 읽어 들인 후 실행 단추나 질의 메뉴의 실행 옵션을 사용하여 실행할 수 있습니다. 명령줄 구분자는 입력 창 옵션 탭에서 설정할 수 있습니다.

배치 파일을 백그라운드에서 실행하려면 입력 창 옵션 탭에서 해당 옵션을 선택합니다.
입력 창을 파일에 저장 입력 창의 현재 내용을 파일에 저장합니다. 나중에 파일을 다시 읽고 실행할 수 있습니다.
출력 버퍼를 파일에 저장

각 결과 집합은 출력 버퍼에 저장됩니다. 이 옵션을 사용하여 출력 버퍼를 파일에 저장할 수 있습니다. 이것은 저장된 데이터를 데이터베이스에 저장하는 것과 매우 유사합니다. 데이터가 표 형식으로 나타나면 출력 창 옵션 탭에서 필드 구분자를 정의할 수 있습니다. 이를 통해 나중에 데이터를 데이터베이스 테이블로 다시 읽어 들일 때 bcp와 함께 사용할 출력 파일을 준비할 수 있습니다. 행 구분자는 항상 줄바꿈입니다.

데이터가 텍스트로 나타날 경우 저장된 데이터는 출력 창에 표시된 것과 동일합니다.
히스토리 버퍼를 파일에 저장 각 실행 명령은 히스토리 아이콘을 누를 때 표시되는 히스토리 버퍼에 저장됩니다. 설명을 추가하는 등 내용을 편집한 후 히스토리 버퍼를 파일에 저장하여 기록 용도나 오류 해결 용도로 사용할 수 있습니다.
ASCII 유형으로 결과 집합 인쇄

현재 결과 집합을 인쇄합니다. 데이터는 ASCII 문자로 나타납니다.

참고:   출력 창에서 열의 크기를 조정하거나 재정렬하는 등 데이터를 변경해도 인쇄된 출력물에는 반영되지 않습니다.
Jtable 유형으로 결과 집합 인쇄
(jdk 1.2에서만)

현재 결과 집합을 인쇄합니다. 데이터는 렌더링된 JTable로 나타납니다.

참고:   출력 창에서 열의 크기를 조정하거나 재정렬하는 등 데이터를 변경해도 인쇄된 출력물에는 반영되지 않습니다.
종료 모든 세션 창과 해당 데이터베이스 연결을 닫고 응용 프로그램 프로세스를 종료합니다.
