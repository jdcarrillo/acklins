개요

jisql은 사용법이 간단한 Transact-SQL (T-SQL) 편집기로서 다음 작업을 할 수 있습니다.
  • 그래픽 사용자 인터페이스(GUI)를 통한 T-SQL 명령문 작성
  • T-SQL 질의 실행
  • 질의 결과를 텍스트 형식이나 표 형식으로 보기
  • 입력 데이터나 결과 데이터를 다른 응용 프로그램으로 가져올 수 있도록 일반적인 파일 형식으로 저장(예: Microsoft Excel 또는 bcp)
jisql은 Java로 작성되었으며 Swing의 일부인 JFC(Java Foundation Classes)를 이용합니다.

jisql은 사용법이 쉽게 설계되었으며, Transact-SQL과 Sybase 데이터베이스 서버에 대한 실무 지식을 가진 사용자를 대상으로 합니다.

연결 창에는 단추와 선택 목록, 입력 창, 출력 창 상태 창, 분리 가능한 메뉴가 있습니다. 창 높이는 창을 분리하는 수평선을 마우스 단추로 끌어 조절할 수 있습니다.

jisql 은 다양한 글꼴로 데이터를 볼 수 있도록 다중 바이트 문자 집합을 지원합니다. 컴퓨터의 환경이 제대로 설정되었는지 반드시 확인해야 합니다. 예를 들어, UNIX의 경우에는 LC_CTYPE과 LANG 환경 변수를 정확하게 설정해야 합니다. 또한 사용할 글꼴이 컴퓨터에 설치되어 있는지도 확인해야 합니다.

jisql 은 입력 창에서 로마 문자 키보드를 사용하여 중국어, 일본어, 한국어 등과 같이 수천개의 문자를 사용하는 언어를 입력할 수 있는 Input Method Framework를 지원합니다.  자세한 내용은 Sun Microsystem's Input Method Framework Overview를 참조하십시오.
