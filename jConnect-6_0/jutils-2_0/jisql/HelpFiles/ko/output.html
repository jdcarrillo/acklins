목차 개요 메뉴 창 단추

출력 창

출력 창에는 명령문을 실행했을 때 서버가 반환하는 결과 집합이 표시됩니다. 출력 창은 Swing Table Component(jTable) 또는 일반 텍스트 영역입니다. 자세한 내용은 표 형식과 텍스트 형식의 출력을 참조하십시오.

결과 집합이 여러 개일 경우(표 형식의 출력만 해당) 각 결과 집합은 출력 창의 개별 탭에 표시됩니다. 다음 결과 집합을 보려면 탭 이름을 누릅니다.

새로운 명령문이나 배치에서 결과 집합이 생성되면 출력 창이 지워집니다.

출력 모드가 표 형식일 경우 일부 데이터 유형은 결과 집합에 표시될 때 특별한 방식으로 처리될 수 있습니다. 특별 처리는 현재 이미지 객체 데이터 유형에 적용됩니다. 자세한 내용은 이미지 객체 열을 참조하십시오.

출력 창 기본 설정을 변경하려면 옵션>옵션 창 열기>출력 창 옵션을 사용하십시오.
