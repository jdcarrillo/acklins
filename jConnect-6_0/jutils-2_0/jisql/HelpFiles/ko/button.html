데이터베이스

도구 모음의 드롭다운 목록에는 도구 모음에 나타난 서버와 관련된 모든 데이터베이스가 표시됩니다. 연결된 세션에서 다른 데이터베이스로 빠르게 변경하려면 드롭다운 목록에서 해당 데이터베이스 이름을 누릅니다. 다른 데이터베이스를 선택하면 히스토리 버퍼에 "[데이터베이스] 사용" 행이 추가됩니다.

입력 창에서 <데이터베이스> 사용 명령을 실행하여 다른 데이터베이스로 변경할 수도 있습니다. 입력 창에서 데이터베이스를 작성, 변경, 삭제해도 도구 모음의 드롭다운 목록은 업데이트되지 않습니다.

히스토리 창을 엽니다. 이 창에는 현재 연결에 대한 이전의 모든 배치(하나 이상의 명령문으로 구성됨)가 표시됩니다. 히스토리 버퍼에 저장할 배치의 최대 수를 지정하려면 옵션 창의 기타 탭을 사용합니다.

재사용할 텍스트를 선택하고 실행을 누르면 해당 명령문이나 배치를 입력 창에 다시 표시하여 재실행할 수 있습니다.
히스토리 창에서 이전 배치를 선택합니다. 선택한 배치는 명령 편집이 가능한 입력 창에 표시하거나, 실행을 눌러 재실행할 수 있습니다.
히스토리 창에서 다음 배치를 선택합니다. 선택한 배치는 명령 편집이 가능한 입력 창에 표시하거나 실행을 눌러 재실행할 수 있습니다.
