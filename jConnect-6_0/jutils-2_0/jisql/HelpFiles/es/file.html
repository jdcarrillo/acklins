Menú Archivo

OPCIÓN
DESCRIPCIÓN
Leer archivo de lote

Permite ejecutar archivos por lotes preparados. Dependiendo del estado definido en la ficha Opciones de la ventana de entrada, es posible leer archivos en dicha ventana, así como modificarlos para, a continuación, ejecutarlos mediante el botón Ir o a la opción Ejecutar del menú Consultar. El delimitador de comando puede configurarse en la ficha Opciones de la ventana de entrada.

Para ejecutar el lote como tarea de fondo, deberá activar la opción adecuada de la ficha Opciones de la ventana de entrada.
Guardar ventana de entrada en archivo Almacena el contenido actual de la ventana de entrada en un archivo. Posteriormente, podrá leerlo y ejecutarlo de nuevo.
Guardar búfer de salida en archivo

Todos los conjuntos de resultados se almacenan en un buffer de salida. Esta opción le permitirá guardar el contenido de este buffer en un archivo, lo que constituye un procedimiento directo para guardar datos almacenados en la base de datos. Si los datos se presentan como elementos de tabla, en la ficha Opciones de la ventana de salida podrá definir un delimitador de campos; lo que permite preparar el archivo de salida para su posterior uso con bcp con el fin de introducir los datos otra vez en una tabla de base de datos. El delimitador de línea es siempre LineFeed.

Si los datos se presentan en forma de texto, se almacenarán exactamente como aparecen en la ventana de salida.
Guardar búfer de historial en archivo Todos los comandos ejecutados se almacenan en la ventana de historial que se abre al hacer clic en el icono Historial. Si lo desea, puede modificar el contenido de esta ventana, añadiendo comentarios, por ejemplo, y guardarlo en un archivo para utilizarlo como registro o para que le sirva de ayuda al resolver problemas.
Imprimir el estilo ASCII del conjunto de resultados

Envía el conjunto de resultados actual a una impresora. Los datos se presentan como texto ASCII.

NOTA:  si modifica los datos en la ventana de salida, (por ejemplo, cambiando el tamaño o reordenando las columnas) estos cambios no aparecerán en la salida impresa.
Imprimir el conjunto de resultados en el estilo JTable
(sólo jdk 1.2)

Envía el conjunto de resultados actual a una impresora. Los datos se presentan como una tabla JTable.

NOTA:  si modifica los datos en la ventana de salida, (por ejemplo, cambiando el tamaño o reordenando las columnas) estos cambios no aparecerán en la salida impresa.
Salir Cierra todas las ventanas de sesión, lo que también hace que se cierren las conexiones con bases de datos asociadas, así como la aplicación.
