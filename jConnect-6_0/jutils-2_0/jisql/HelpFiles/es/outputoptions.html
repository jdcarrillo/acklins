Opciones de la ventana de salida

Estas opciones, que dependen de la conexión empleada, constituyen un medio eficaz de configurar los parámetros de la ventana de salida. Haga clic en Aplicar para guardar los cambios.

OPCIÓN DESCRIPCIÓN
El conjunto de resultados se presentará con el siguiente formato (Texto/Tabla)

Determina el modo en que se presentarán los conjuntos de resultados siguientes en la ventana de salida. El conjunto de resultado actual no se verá afectado.

El valor por defecto es "Tabla".

Ver también: Salida en formato tabla o en formato texto.

Método utilizado en el objeto DB para mostrar los datos Método utilizado para presentar valores de datos almacenados en un objeto Java que se encuentra en la base de datos. El valor por defecto es "toString", pero puede cambiarlo por el nombre de un método definido por usted.
Extraer automáticamente los datos del objeto Java DB Seleccione esta casilla para extraer automáticamente los datos de los objetos Java almacenados en la base de datos.
Delimitador de campo a usar en archivo de salida (sólo tabla) Define el carácter que se utilizará como delimitador entre los datos de los campos cuando se guarde un conjunto de resultados en un archivo. Esta opción sólo es válida cuando el conjunto de resultados se presenta en modo tabla. Los conjuntos de resultados presentados en modo texto aparecerán exactamente igual en el archivo de salida que en la ventana de salida.
