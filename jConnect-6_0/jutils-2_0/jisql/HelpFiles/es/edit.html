OPCIÓN
DESCRIPCIÓN
Cortar

Permite cortar contenido de otras aplicaciones Java para pegarlo en jisql. Asimismo, se puede cortar contenido de jisql y pegarlo en otras aplicaciones Java, siempre y cuando la función Pegar sea compatible con dicha aplicación.
Copiar Permite copiar contenido de otras aplicaciones Java para pegarlo en jisql. Asimismo, se puede copiar contenido de jisql y pegarlo en otras aplicaciones Java, siempre y cuando la función Pegar sea compatible con dicha aplicación.
Pegar Permite pegar contenido en la ventana de entrada.
NOTA cortar, copiar y pegar entre jisql y aplicaciones nativas, como XTerm o MS Word, funciona en plataformas Microsoft (incluidos los métodos abreviados de teclas Ctrl -c y -v). En las plataformas UNIX es probable que sólo funcione en administradores de ventanas basados en Motif, como CDE y mvm.
