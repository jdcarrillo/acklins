Base de datos

El cuadro de lista desplegable de la barra de herramientas muestra todas las bases de datos asociadas con el servidor cuyo nombre aparece en dicha barra. Para cambiar rápidamente de una base de datos a otra en el curso de una sesión, haga clic en el nombre de la base deseada en este cuadro de lista. Cuando utilice una base de datos distinta, se añadirá una línea "use [base de datos]" al buffer de historial.

Asimismo, se puede cambiar de la base de datos actual a otra ejecutando un comando use <base de datos> en la ventana de entrada. Al crear, cambiar o eliminar una base de datos en la ventana de entrada, la lista desplegable de la barra de herramientas no se actualizará.

Haga clic en el icono Historial para abrir una ventana de historial que muestre todos los lotes ejecutados (formados por una o más instrucciones) de la conexión actual. Para determinar el número máximo de lotes que se debe almacenar en el buffer de historial, utilice la ficha Varios de la ventana de opciones de jisql.

Resalte el texto que desee volver a utilizar y haga clic en Ir para que la instrucción o lote de instrucciones se vuelva a visualizar en la ventana de entrada y se ejecute de nuevo.
Haga clic en el icono de flecha hacia arriba para seleccionar el lote anterior de la ventana de historial de modo que aparezca en la ventana de entrada, en la que podrá modificar el comando o hacer clic en Ir para ejecutarlo de nuevo.
Haga clic en el icono de flecha hacia abajo para seleccionar el lote siguiente de la ventana de historial de modo que aparezca en la ventana de entrada, en la que podrá modificar el comando o hacer clic en Ir para ejecutarlo de nuevo.
