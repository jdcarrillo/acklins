Presentación

jisql es un editor Transact-SQL (T-SQL) fácil de utilizar que le ayudará a realizar las siguientes tareas:
  • Componer instrucciones de comandos T-SQL mediante una interfaz de usuario gráfica.
  • Ejecutar consultas T-SQL.
  • Ver el resultado de las consultas en forma de texto o en una tabla.
  • Guardar los datos que introduzca o los de resultado en formatos de archivo comunes que puedan importarse desde otras aplicaciones, como por ejemplo, Microsoft Excel o bcp.
jisql está escrito en lenguaje Java y aprovecha las ventajas que ofrecen las JFC (Java Foundation Classes, clases básicas de Java), que forman parte de Swing.

jisql se ha concebido de forma que su utilización resulte intuitiva para el usuario y da por supuesto que éste tiene unos conocimientos básicos de trabajo con Transact-SQL y los servidores de bases de datos Sybase.

La ventana de conexión contiene botones y listas de selección , la ventana de entrada, la ventana de salida, la ventana de estado y la barra de menús flotante. La altura de las ventanas puede modificarse de forma individual arrastrando las líneas horizontales que las separan.

jisql es compatible con los juegos de caracteres multibyte, lo que permite mostrar los datos en distintas fuentes. Sin embargo, para ello, es necesario comprobar que el entorno de la máquina esté debidamente configurado; por ejemplo, en UNIX hay que prestar una especial atención a las variables de entorno LC_CTYPE y LANG. Asimismo, es preciso verificar que las fuentes que se desea utilizar estén instaladas en la máquina.

jisql es compatible con el sistema de métodos de introducción de caracteres (Input Method Framework) de Sun, que permite escribir texto con un teclado de caracteres romanos en la ventana de entrada tanto en chino como en japonés o coreano, idiomas que utilizan miles de caracteres distintos.  Si desea obtener más información al respecto, consulte la página Input Method Framework Overview del sitio web de Sun Microsystems.
