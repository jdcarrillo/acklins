Bienvenido a jisql

jisql le brinda la oportunidad de interactuar con un servidor de bases de datos Sybase; de hecho, desempeña las mismas tareas que isql, pero aporta funciones adicionales y una interfaz de usuario gráfica.

jisql incluye los siguientes temas en línea:

  • Presentación
  • Tareas
    • Introducción y ejecución de consultas
    • Nueva utilización de consultas ya ejecutadas
  • Ventanas
    • Ventana de entrada
    • Ventana de historial
    • Ventana de salida
    • Ventana de estado
    • Ventanas de error
  • Menús
    • Archivo
    • Edición
    • Borrar
    • Conexión
    • Consultar
    • Lista
    • Opciones
      • Opciones de la ventana de salida
      • Opciones de la ventana de entrada
      • Opciones de bases de datos
      • Diagnósticos de bases de datos
      • Opciones varias
    • Ayuda
  • Botones y listas de selección
  • Problemas conocidos
