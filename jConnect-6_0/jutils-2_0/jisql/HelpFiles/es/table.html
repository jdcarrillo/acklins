Salida en formato tabla o en formato texto

Los conjuntos de resultados pueden presentarse en un componente Swing denominado JTable o en un área de texto normal. Para seleccionar cualquiera de estos formatos se utiliza el panel Opciones de la ventana de salida del menú Opciones.

Las ventajas de utilizar el formato JTable son las siguientes:

  • Apariencia más atractiva.
  • Mejor salida de los datos a un archivo, incluida la preparación de datos para formato bcp.
  • Posibilidad de arrastrar y colocar los encabezados de las columnas para reorganizarlas.
  • Gestión eficaz de las columnas de imágenes (consulte Columnas de imágenes).
  • Gestión eficaz de las columnas de objetos Java.
  • Cuando existen varios conjuntos de resultados, cada uno se muestra en una ficha independiente.

Las ventajas de utilizar un área de texto son las siguientes:

  • Salida más rápida.
