Ventana de salida

La ventana de salida muestra los conjuntos de resultados devueltos por el servidor como respuesta a la ejecución de una instrucción. Este resultado puede adoptar la forma de un componente de tabla Swing (jTable) o de un área de texto normal. Si desea más información, consulte Salida en formato tabla o en formato texto.

Si existen varios conjuntos de resultados (sólo en la salida en formato tabla), cada uno de ellos se muestra en una "ficha" independiente en la ventana de salida. Haga clic en el título de la ficha para ver el conjunto de resultados deseado.

La ventana de salida se borra cuando llega el conjunto de resultados de una nueva instrucción o lote de instrucciones.

Si el modo de salida es una tabla, algunos tipos de datos pueden gestionarse de forma especial cuando aparecen en el conjunto de resultados. Por ejemplo, esta gestión especial está instaurada para el tipo de datos objeto de imágenes. Para más información, consulte también Columnas de imágenes.

Para cambiar los valores por defecto de la ventana de salida, utilice Opciones>Abrir ventana de opciones>Opciones de ventana de salida.
