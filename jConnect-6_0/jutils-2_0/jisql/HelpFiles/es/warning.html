Errores y advertencias SQL

Los errores y advertencias de SQL normalmente contienen mensajes de información provenientes de la base de datos. Por defecto, cada advertencia hace aparecer una ventana independiente. Sin embargo, estas advertencias pueden configurarse para que se presenten en la ventana de estado o para que no aparezcan en absoluto.
