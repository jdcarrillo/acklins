Ventanas de error

Las ventanas de error aparecen cuando la base de datos, jConnect, o Java VM produce una advertencia o un error SQL. Estas ventanas incluyen un botón "Mostrar stacktrace" que abre a su vez otra ventana con la stacktrace. Si lo desea, puede guardar el error en un archivo para que le sirva de ayuda al investigar el problema.
