Ventanas

jisql posibilita tener abiertas ninguna, una o varias conexiones a la vez, cada una de ellas con su propia ventana de sesión. Cada nueva ventana de sesión contiene los siguientes elementos:

  • Una ventana de entrada: en la que se introducen las instrucciones de las consultas.
  • Una ventana de salida: que muestra los resultados de las consultas ejecutadas.
  • Una ventana de estado: ubicada debajo de la ventana de salida, y que muestra los errores de ejecución de las consultas y los mensajes de información.
  • La barra de herramientas: que contiene el botón Ir para ejecutar las consultas, una lista de selección con las bases de datos que residen en el servidor con el que se está conectado, y los iconos historial, que ayudan a visualizar consultas ejecutadas con anterioridad.
