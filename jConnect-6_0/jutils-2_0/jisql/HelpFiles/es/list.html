Listar tablas de usuario

Lista las tablas de usuario de la base de datos actual. Haga doble clic en el nombre de una tabla para mostrar sus columnas, incluida información sobre la definición de la tabla, como el nombre de las columnas el tipo de datos que contienen y la longitud máxima del tipo de datos que pueden incluir.

Si desea crear una instrucción Select para una tabla o columna determinada, haga clic en los nombres de tabla o columna y haga clic en Seleccionar. Para elegir varios nombres que no estén seguidos, mantenga pulsada la tecla Ctrl y haga clic en cada uno de los nombres deseados. Si quiere seleccionar varios nombres consecutivos, mantenga pulsada la tecla Mayús y haga clic en el primer y último nombre del grupo. En la ventana de entrada, jisql insertará una instrucción Select para las tablas o columnas elegidas.

Listar Procedimientos almacenados

Lista los procedimientos almacenados de la base de datos seleccionada actualmente. Haga clic en el nombre de un procedimiento almacenado y, a continuación, haga clic en Mostrar texto para visualizar el texto que contiene, que podrá modificar o guardar en un archivo.

Si desea utilizar parte del texto en una consulta, resáltelo, pulse Ctrl+c, sitúe el cursor en la ventana de entrada y, a continuación, pulse Ctrl+v para que jisql inserte el texto copiado en la ventana. Esta operación no funciona con todos los administradores de ventanas. Para ejecutar la instrucción, haga clic en Ir.
