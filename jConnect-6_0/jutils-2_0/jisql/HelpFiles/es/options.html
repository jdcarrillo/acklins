Menú Opciones

La opción de este menú abre la ventana de opciones de jisql, que permite configurar nuevos valores para la ventana de salida, la ventana de entrada, la base de datos, los diagnósticos de bases de datos, así como opciones varias.

Estos valores dependen de la conexión, pero es posible que no muestren el valor activo actual para una conexión determinada. La finalidad de estas opciones es poner al alcance del usuario un modo eficaz de configurar los valores de las opciones.

Aplicar nuevos valores a las opciones sólo afecta a la ventana de sesión "activa". Si hay más de una ventana de sesión abierta, la ventana activa es la que se encuentra delante de las otras ventanas de sesión y cuya barra de título es de otro color o de color más intenso.
