Ausgabefenster

Das Ausgabefenster zeigt die Ergebnismengen, die vom Server nach dem Abarbeiten einer Anweisung zurückgegeben werden. Es kann sich um eine Swing-Tabellenkomponente (jTable) oder einen normalen Textbereich handeln. Siehe dazu auch Ausgabe als Tabelle oder Text.

Wenn mehrere Ergebnismengen vorhanden sind (nur bei tabellarischer Ausgabe), wird jede Ergebnismenge in einem eigenen "Register" des Ausgabefensters angezeigt. Klicken Sie auf den Titel des Registers, um die nächste Ergebnismenge anzuzeigen.

Das Ausgabefenster wird geleert, wenn die Ergebnismenge von einer neuen Anweisung oder Anweisungsfolge kommt.

Wenn die Ausgabeform auf tabellarisches Format gesetzt ist, können bestimmte Datentypen auf besondere Weise behandelt werden, wenn sie in der Ergebnismenge erscheinen. Eine Spezialverarbeitung ist derzeit für die Bilddatentypen vorgesehen (image). Siehe dazu auch Image-Objektspalten.

Um die Standardwerte des Ausgabefenster zu ändern, benutzen Sie Optionen>Öffne Fenster für Optionen>Optionen für das Ausgabefenster.
