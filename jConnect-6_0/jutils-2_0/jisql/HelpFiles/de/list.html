Zeige Benutzertabellen

Damit werden die Benutzertabellen der aktuellen Datenbank aufgelistet. Doppelklicken Sie auf einen Tabellennamen, um die Spalten dieser Tabelle anzuzeigen, einschließlich Teile der Tabellendefinition, wie z.B. Spaltenname, Datentyp und Länge des Datentyps.

Wenn Sie eine Anweisung für eine bestimmte Tabelle oder Spalte erstellen wollen, klicken Sie auf die Namen einer Tabelle oder Spalte und dann auf Auswählen. Um mehrere, nicht nebeneinander liegende Namen zu wählen, halten Sie die Strg-Taste, während Sie auf die einzelnen Namen klicken. Um mehrere nebeneinander liegende Namen zu wählen, klicken Sie auf den ersten Namen, halten dann die Umschalttaste gedrückt und klicken auf den letzten Namen in der Reihe. jisql fügt eine Auswahlanweisung für die gewählten Tabellen oder Spalten im Eingabefenster ein.

Zeige gespeicherte Prozeduren

Damit werden die gespeicherten Prozeduren in der derzeit gewählten Datenbank angezeigt. Klicken Sie auf den Namen einer gespeicherten Prozedur und dann auf Text zeigen, um den Text der ausgewählten Prozedur anzuzeigen, den Sie dann in einer Datei speichern und bearbeiten können.

Wenn Sie einen Teil des Textes in einer Abfrage verwenden wollen, markieren Sie den Text, drücken Strg+c, setzen den Cursor in das Eingabefenster und drücken dann Strg+v. jisql fügt den kopierten Text in das Fenster ein. Klicken Sie auf Go, um die Anweisung auszuführen. Dies funktioniert aber nicht in allen Fenstermanagern.
