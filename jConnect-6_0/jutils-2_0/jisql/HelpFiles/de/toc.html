Willkommen bei jisql!

Sie können jisql aufrufen, um einen Dialog mit einem Sybase-Datenbankserver aufzunehmen. Die Anwendung erfüllt denselben Zweck wie das bekannte Dialogprogramm isql, bietet aber zusätzliche Funktionen und eine grafische Benutzeroberfläche.

Für jisql stehen folgende Hilfethemen zur Verfügung:

  • Überblick
  • Aufgaben
    • Abfragen eingeben und ausführen
    • Vorherige Anweisungen wieder aufrufen
  • Fenster
    • Eingabefenster
    • Ablauffenster
    • Ausgabefenster
    • Statusfenster
    • Fehler
  • Menüs
    • Datei
    • Bearbeiten
    • Löschen
    • Verbindung
    • Anweisung
    • Inhalt
    • Optionen
      • Optionen für das Ausgabefenster
      • Optionen für das Eingabefenster
      • Datenbankoptionen
      • Datenbankdiagnose
      • Sonstiges
    • Hilfe
  • Schaltflächen und Auswahllisten
  • Bekannte Probleme
