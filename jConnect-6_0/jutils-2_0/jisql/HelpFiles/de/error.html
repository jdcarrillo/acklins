Fehleranzeigen

Fehler werden angezeigt, wenn eine SQL-Ausnahmebedingung oder eine Warnung von der Datenbank, von jConnect oder von der Java VM gemeldet wird. Das Fehlerfenster enthält eine Schaltfläche mit der Bezeichnung "Zeige Fehlermeldungen", mit der ein anderes Fenster geöffnet wird, in dem die Fehlermeldung erscheint. Wenn erforderlich, können Sie die Fehlermeldung in einer Datei speichern, um die Ursache herauszufinden.
