Ausgabe als Tabelle oder Text

Ergebnismengen können in einer Swing-Komponente namens Jtable oder in einem normalen Textbereich angezeigt werden. Sie können diese Formate mit Optionen für das Ausgabefenster im Menü "Optionen" wählen.

Vorteile von JTable:

  • Bessere Darstellung
  • Bessere Ausgabe von Daten in eine Datei, mit korrekter Aufbereitung der Daten für bcp
  • Tabellentitel können gezogen und abgelegt werden, um die Spalten neu zu ordnen
  • Einfache Behandlung von Tabellenspalten (siehe Bildpalten)
  • Einfache Bearbeitung von Java-Objektspalten
  • Wenn mehrere Ergebnismengen vorhanden sind, wird jede Ergebnismenge in einem eigenen Register angezeigt.

Vorteile der Ausgabe in einem Textbereich:

  • Schnellere Ausgabe
