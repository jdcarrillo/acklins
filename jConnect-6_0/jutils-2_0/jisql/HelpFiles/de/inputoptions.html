Optionen für das Eingabefenster

  • Abschlussbefehl einer Anweisung
    Der Abschlussbefehl markiert das Ende einer Anweisung und sendet die darin enthaltenen Befehle an den Datenbankserver, damit sie dort abgearbeitet werden. Normalerweise wird "go", der Begrenzer für Sybase Transact-SQL, verwendet.

    Wenn Sie einen anderen Abschluss als "go" verwenden wollen, geben Sie ihn hier ein.

    HINWEIS: Leerzeichen werden entfernt. Daher sind ' go ' und 'go' identisch.

  • Anzeige der Datei im Eingabefenster?
    Wenn Sie die Standardvorgabe verwenden (markiert), wird eine importierte Datei mit einer Anweisungsfolge im Eingabefenster angezeigt, damit Sie sie bearbeiten und ausführen können, indem Sie die Schaltfläche Go anklicken oder im Menü "Abfrage" die Option Ausführen aufrufen. Wenn das Kontrollkästchen nicht aktiviert ist, wird die Anweisungsdatei im Hintergrund ausgeführt. Die Ausgabe bzw. die Statusmeldungen werden in beiden Fällen angezeigt.

  • Anweisungen abbrechen bei Fehlermeldung?

    Wenn diese Option markiert ist (Standardvorgabe) und eine SQL-Ausnahmebedingung auftritt, während eine lange Anweisungsfolge läuft, bricht jisql die Anweisungsfolge ab.
