Datenbank

Die Dropdown-Liste in der Symbolleiste zeigt alle Datenbanken, die mit dem in der Symbolleiste angezeigten Server verbunden sind. Um innerhalb einer Verbindungssitzung rasch von einer Datenbank in eine andere zu wechseln, klicken Sie auf den Datenbanknamen in der Dropdown-Liste. Wenn Sie eine andere Datenbank wählen, wird die Zeile "use [Datenbank]" in den Ablaufpuffer geschrieben.

Sie können auch aus der aktuellen Datenbank in eine andere wechseln, indem Sie den Befehl use <Datenbank> aus dem Eingabefenster ausführen. Wenn Sie eine Datenbank im Eingabefenster erstellen, ändern oder löschen, wird die Dropdown-Liste, in der die Datenbanken aufgelistet werden, nicht aktualisiert.

Klicken Sie auf das Symbol für die Ablaufverfolgung, um ein Ablauffenster anzuzeigen, in dem alle früheren Anweisungsfolgen (1 oder mehr Anweisungen) für die aktuelle Verbindung enthalten sind. Um festzulegen, wieviele Anweisungsfolgen maximal im Ablaufpuffer gespeichert werden sollen, wählen Sie das Register Sonstiges im Fenster Optionen.

Markieren Sie den Text, den Sie wiederverwenden wollen, und klicken Sie auf Go, um die Anweisung oder Anweisungsfolge im Eingabefenster anzuzeigen und nochmals auszuführen.

Klicken Sie auf das Aufwärtssymbol, um die vorherige Anweisungsfolge im Ablauffenster zu wählen und im Eingabefenster anzuzeigen. Danach können Sie die Befehle bearbeiten oder einfach auf Go klicken, um sie nochmals auszuführen.
Klicken Sie auf das Abwärtssymbol, um die nächste Anweisungsfolge im Ablauffenster zu wählen und im Eingabefenster anzuzeigen. Danach können Sie die Befehle bearbeiten oder einfach auf Go klicken, um sie nochmals auszuführen.
