Eingabefenster

Geben Sie eine oder mehrere Abfrageanweisungen in das Eingabefenster ein oder verwenden Sie dazu die Option "Einfügen". Sie müssen möglicherweise die Anweisungen durch den Befehlsbegrenzer trennen (Standard ist 'go', Sie können dies aber mit Optionen>Öffne Fenster für Optionen>Optionen für das Eingabefenster ändern). Das Eingabefenster ist das einzige Ziel für Einfügeoperationen mit dem Befehl im Menü Bearbeiten.

Um die Standardwerte des Eingabefensters zu ändern, verwenden Sie Optionen>Öffne Fenster für Optionen>Optionen für das Eingabefenster.
