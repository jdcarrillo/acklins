Abfragen eingeben und ausführen

Eine Anweisungsfolge (Skript) ist eine Sammlung von Transact SQL-Anweisungen, um eine bestimmte Aufgabe auszuführen. Skripten, die als Dateien gespeichert werden, bilden die Grundlage für viele Server-Anwendungsprojekte. Definitionen von Tabellen, Indizes und gespeicherte Prozeduren, sowie Reports und Anweisungsfolgen werden normalerweise in Skript-Dateien gespeichert.

Wenn Sie eine Verbindung mit einem Server herstellen, klicken Sie in der Dropdown-Liste auf die Datenbank, mit der Sie arbeiten möchten. Sie können Abfragen auf unterschiedliche Arten eingeben. Um eine Abfrage auszuführen, klicken Sie auf die Schaltfläche Go in der Symbolleiste oder wählen "Anweisung ausführen" aus dem Menü "Anweisung".

  • Sie können Ihre Abfrage direkt in das Eingabefenster eingeben und klicken dann auf Go.
  • Sie können eine Option aus dem Menü Inhalt wählen..
    Wenn die Liste Benutzertabellen oder Systemtabellen enthält, klicken Sie auf den Namen einer Tabelle oder Spalte und auf Auswählen. Um mehrere, nicht nebeneinander liegende Namen zu wählen, halten Sie die Strg-Taste, während Sie auf die einzelnen Namen klicken. Um mehrere nebeneinander liegende Namen zu wählen, klicken Sie auf den ersten Namen, halten dann die Umschalttaste gedrückt und klicken auf den letzten Namen in der Reihe. jisql fügt eine Auswahlanweisung für die gewählten Tabellen oder Spalten im Eingabefenster ein.
    Wenn gespeicherte Prozeduren oder Trigger gelistet sind, klicken Sie auf den Namen des gewünschten Elements, das Sie anzeigen wollen, und auf Text zeigen. Ein Fenster wird geöffnet, in dem der Text der gespeicherten Prozedur oder des Triggers angezeigt wird. Wenn Sie einen Teil des Textes in einer Abfrage verwenden wollen, markieren Sie den Text, drücken Strg+c, setzen den Cursor in das Eingabefenster und drücken dann Strg+v. jisql fügt den kopierten Text in das Fenster ein (nicht bei allen Window-Managern).
  • Markieren Sie eine frühere Anweisung im Eingabefenster und klicken Sie auf Go oder wählen Sie "Anweisung ausführen" aus dem Menü "Anweisung".
  • Übertragen Sie die Anweisungen aus dem Ablauffenster in das Eingabefenster. (Sie müssen mindestens eine Anweisung oder eine Anweisungsfolge ausgeführt haben, damit im Ablauffenster etwas erscheint.)

    Klicken Sie auf das Symbol für das Ablaufprotokoll . Im Ablauffenster markieren Sie die Anweisungen, die Sie ausführen wollen, und klicken dann auf Go in der Symbolleiste des Verbindungsfensters. Sie können auch Strg+c verwenden, um Anweisungen aus dem Ablauffenster zu kopieren und mit Strg+v in das Eingabefenster einzufügen.

Die Ergebnismengen werden im Ausgabefenster angezeigt.
