Statusfenster

Das Statusfenster zeigt unterschiedliche Meldungen an:

  • Meldungen über den erfolgreichen Abschluß von Anweisungen, die keine Ergebnismenge liefern (zum Beispiel create oder update)

  • Warnmeldungen, die in das Statusfenster umgeleitet werden, indem die entsprechende Option in den Einstellungen für Sonstiges aktiviert wird

  • "x rows affected (x Zeilen betroffen)", wenn diese Option aktiviert wurde.
