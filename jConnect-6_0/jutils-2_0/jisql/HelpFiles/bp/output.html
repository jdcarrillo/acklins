Janela de Saída

A Janela de Saída exibe os conjuntos de resultados retornados pelo servidor em resposta à execução de uma instrução. Este pode ser um Componente de Tabela Swing (jTable) ou uma área de texto comum. Vide Saída de Tabela vs. Texto para mais informações.

Quando você tem múltiplos ResultSets (apenas saída de Tabela), cada ResultSet é exibido numa "guia" separada na Janela de Saída. Clique no título da guia para exibir o ResultSet seguinte.

A Janela de Saída é limpa quando o ResultSet chega de uma nova instrução ou lote.

Quando o modo de saída é uma Tabela, alguns tipos de dados podem ser tratados de maneira especial quando aparecem no ResultSet. Um tratamento especial está atualmente implementado para o tipo de dados objeto de imagem. Vide também colunas de objetos de imagem para mais informações.

Para mudar os padrões da Janela de Saída, use Opções>Abrir Janela de Opção>Opções da Janela de Saída.
