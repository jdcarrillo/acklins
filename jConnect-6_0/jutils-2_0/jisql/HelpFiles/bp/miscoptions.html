Clique em Aplicar para salvar suas mudanças.
 
OPÇÃO DESCRIÇÃO
Avisos SQL devem ser apresentados Especifica como você quer que as Exceções ou avisos SQL sejam exibidos:
  • Numa janela de texto separada 
  • Numa Janela pop-up
  • Na Janela de Status (embaixo da Janela de Saída)
  • De jeito nenhum
O padrão é "de jeito nenhum."
Máximo de instruções no histórico Especifica o número de instruções que o jisql deve armazenar no buffer de histórico. Uma instrução pode consistir em uma linha ou em várias linhas num lote. O padrão é 50. Digite zero (0) para ilimitado.
Definir Fonte Use a caixa de listagem suspensa para mudar o fonte e/ou o tamanho usado na janela de sessão ativa. Os fontes para texto de etiqueta e de borda vão mudar, mas seus tamanhos de fonte permanecerão os mesmos. Se uma combinação de estilo de fonte e tamanho não funcionar conjuntamente, os caracteres não serão exibidos.
