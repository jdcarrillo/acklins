Opções da Janela de Saída

Estas opções oferecem um jeito conveniente de definir os parâmetros da Janela de Saída. Estas opções são específicas para a conexão. Clique em Aplicar para salvar as mudanças.

OPÇÃO DESCRIÇÃO
Apresentar ResultSet como (Texto/Tabela)

Especifica como apresentar futuros ResultSets na Janela de Saída. O atual ResultSet não é afetado.

O padrão é "Tabela."

Vide também: Saída de Tabela vs Texto.

Método usado no DB Object para exibir dados O método usado para imprimir do valores de dados armazenados num objeto Java que esteja armazenado na base de dados. O padrão é "toString,", mas você pode mudá-lo para um nome de método definido pelo usuário.
Extrair automaticamente os dados do Objeto Java da base de dados Marque esta caixa para extrair automaticamente os dados de objetos Java armazenados na base de dados.
Delimitador de campo a ser usado no arquivo de saída (apenas tabela) Define o caractere a ser usado como delimitador entre os campos de dados quando um ResultSet é escrito num arquivo. Isto só é válido quando o ResultSet é apresentado numa Tabela. Os ResultSets apresentados em modo Texto aparecem exatamente da mesma forma no arquivo de saída e na Janela de Saída.
