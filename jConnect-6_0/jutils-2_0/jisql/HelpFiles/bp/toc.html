Bem-vindo ao jisql!

O jisql dá a você a oportunidade de interagir com um servidor de base de dados Sybase. Ele tem o mesmo objetivo que o isql, mas lhe oferece recursos adicionais e uma interface gráfica de usuário.

O jisql oferece os seguintes tópicos on-line:

  • Visão Geral
  • Tarefas
    • Digitar e executar consultas
    • Chamar novamente consultas anteriores
  • Janelas
    • Entrada
    • Histórico
    • Saída
    • Status
    • Erros
  • Menus
    • Arquivo
    • Editar
    • Limpar
    • Conexão
    • Consulta
    • Listar
    • Opções
      • Opções da Janela de Saída
      • Opções da Janela de Entrada
      • Opções de Base de Dados
      • Diagnóstico de Base de Dados
      • Diversas
    • Ajuda
  • Botões e Listas de Seleção
  • Problemas Conhecidos
