Visão Geral

O jisql é um editor Transact-SQL (T-SQL) de fácil utilização, que ajuda você a:
  • Compor instruções de comando T-SQL usando uma interface gráfica de usuário
  • Executar consultas T-SQL
  • Visualizar os resultados da consulta como texto ou numa tabela
  • Salvar seus dados de saída ou de resultado em formatos de arquivo comuns que podem ser importados por outros aplicativos (por exemplo, Microsoft Excel ou bcp)
O jisql é escrito em Java e tira proveito das Java Foundation Classes (JFC), que são parte do Swing.

O jisql foi criado para ser razoavelmente intuitivo e pressupõe que você tenha um conhecimento prático de Transact-SQL e dos servidores de base de dados Sybase.

A janela de conexão contém botões e listas de seleção, a Janela de Entrada, a Janela de Saída , a Janela de Status e o menu destacável. A altura das janelas pode ser redimensionada individualmente, arrastando-se a linha horizontal que as separa.

O jisql aceita conjuntos de caracteres multibyte para exibir dados em diferentes fontes. Você deve verificar se o ambiente do computador está configurado corretamente; por exemplo, o UNIX exige que se preste uma atenção especial às variáveis de ambiente LC_CTYPE e LANG. Você também deve verificar se as fontes que você quer usar estão realmente instaladas em seu computador.

O jisql aceita o Input Method Framework, o que torna possível digitar textos na Janela de Entrada em Chinês, Japonês ou Coreano - línguas que usam milhares de caracteres diferentes - num teclado de caracteres romanos.  Para mais informações, vide Visão Geral do Input Method Framework da Sun Microsystems .
