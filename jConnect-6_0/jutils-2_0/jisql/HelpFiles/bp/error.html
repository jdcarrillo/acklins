Janelas de Erro

As janelas de Erro aparecem quando uma Exceção de SQL ou aviso é emitido pela base de dados, jConnect ou pelo Java VM. A janela de Erro contém um botão "Mostrar rastreio de pilha" que abre outra janela para mostrar o rastreio de pilha. Se necessário, você pode salvar o rastreio de pilha num arquivo, para ajudar a investigar o problema.
