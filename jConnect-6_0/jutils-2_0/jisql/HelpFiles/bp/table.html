Saída em formato tabela ou em formato texto

Os ResultSets podem ser apresentados num componente Swing chamado JTable ou numa área de texto normal. Você pode escolher o formato usando o Painel Opção de Janela de Saída do menu Opções.

As vantagens de se usar uma JTable são:

  • Melhor aparência
  • Melhor saída de dados num arquivo, inclusive preparação de dados de tipo bcp
  • Cabeçalhos de tabelas podem ser arrastados e soltos para reordenar as colunas
  • Tratamento conveniente de colunas de imagens (vide Colunas de Imagens)
  • Tratamento conveniente de colunas de objetos Java
  • Quando você tem muitos ResultSets, cada ResultSet é exibido numa guia separada

As vantagens de se usar uma área de texto são:

  • Saída mais rápida
