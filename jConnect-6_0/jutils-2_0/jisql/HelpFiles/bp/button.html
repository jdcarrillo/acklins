Base de Dados

A caixa de listagem suspensa na barra de ferramentas mostra todas as bases de dados associadas ao servidor nomeado na barra de ferramentas. Para passar rapidamente de uma base de dados para outra em meio a uma sessão de conexão, clique no nome da base de dados na caixa de listagem suspensa. Quando você selecionar uma base de dados diferente, uma linha "use [base de dados]" é adicionada ao buffer histórico.

Você também pode passar da base de dados atual para outra diferente executando um comando use <base de dados> na Janela de Entrada. Quando você cria, muda ou cancela uma base de dados na Janela de Entrada, a lista suspensa de base de dados na barra de ferramentas não se atualiza.

Clique no ícone Histórico para abrir uma Janela de Histórico, que exibe todos os lotes anteriores (que consistem em 1 ou várias instruções) para a conexão atual. Para especificar o número máximo de lotes a armazenar no buffer histórico, use a guia Diversas na Janela Opções.

Realce o texto que você quer reutilizar e clique em Ir para reexibir a instrução ou lote na Janela de Entrada e execute-a de novo.
Clique no ícone Acima para selecionar o lote anterior na Janela de Histórico e exiba-o na Janela de Entrada, onde você pode editar o comando, ou clique em Ir e execute-o de novo.
Clique no ícone Abaixo para selecionar o próximo lote na Janela de Histórico e exiba-o na Janela de Entrada, onde você pode editar o comando, ou clique em Ir e execute-o de novo.
