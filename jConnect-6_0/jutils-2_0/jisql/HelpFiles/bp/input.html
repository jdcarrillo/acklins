Janela de Entrada

Digite ou cole uma ou mais instruções de consulta na Janela de Entrada. Você pode dividir as instruções pelo delimitador de comandos (o padrão é 'go', alterável usando Opções>Abrir Janela de Opção>Opções da Janela de Entrada). A Janela de Entrada é o único alvo para uma operação de colagem executada a partir do menu Editar.

Para modificar os padrões da Janela de Entrada, use Opções>Abrir Janela de Opção>Opções da Janela de Entrada.
