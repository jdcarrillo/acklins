Menu Editar

OPÇÃO
DESCRIÇÃO
Recortar

Permite-lhe Recortar de outros aplicativos Java e Colar no jisql. Você também pode Recortar do jisql para outro aplicativo Java se a função Recortar for aceita nesse aplicativo.
Copiar Permite-lhe Copiar de outros aplicativos Java e Colar no jisql. Você também pode Copiar do jisql para outros aplicativo Java se a função Colar for aceita nesse aplicativo.
Colar Permite-lhe colar trabalho dentro da Janela de Entrada.
OBSERVAÇÃO Recortar, Copiar e Colar entre o jisql e os aplicativos originais (como XTerm ou MS Word) são funções que funcionam em plataformas Microsoft (que incluem os atalhos Ctrl -c e-v). Em plataformas UNIX, talvez funcionem apenas em Gerenciadores de Janela baseados em Motivo, como CDE e mvm.
