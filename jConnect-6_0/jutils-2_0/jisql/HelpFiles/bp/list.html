DESCRIÇÃO
Listar Tabelas do Usuário

Enumera as tabelas do usuário da base de dados atual. Clique duas vezes no nome da tabela para exibir as colunas dessa tabela, inclusive partes da definição da tabela, como o nome de coluna, tipo de dados e comprimento do tipo de dados.

Para criar uma instrução de seleção para uma determinada tabela ou coluna, clique no(s) nome(s) de uma tabela ou coluna e clique em Selecionar. Para escolher diversos nomes não contíguos, mantenha pressionada a tecla Control e clique em cada nome que queira selecionar. Para escolher vários nomes contíguos, mantenha pressionada a tecla Shift e clique no primeiro e no último nome do grupo de nomes que você queira selecionar. O jisql insere uma instrução de seleção para as tabelas e colunas que você escolheu na Janela de Entrada.

Listar Procedimentos Armazenados

Enumera os procedimentos armazenados na base de dados presentemente selecionada. Clique no nome de um procedimento armazenado e em seguida clique em Mostrar texto para exibir o texto do procedimento selecionado, que você pode salvar num arquivo e editar.

Se você quiser usar parte do texto numa consulta, realce o texto, pressione Ctrl+c, coloque seu cursor na Janela de Entrada e em seguida pressione Ctrl+v. O jisql insere o texto copiado na janela. Clique em Ir para executar a instrução. Porém, isso não funciona com todos os Gerenciadores de Janela.
