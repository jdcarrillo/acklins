Menu Opções

A opção deste menu abre a Janela Opções do jisql , a qual lhe permite definir novos valores para a Janela de Saída, para a Janela de Entrada, para a base de dados, diagnóstico de base de dados e diversas.

Esses valores são específicos para cada conexão, mas talvez não mostrem o valor atual real para uma conexão. O propósito dessas opções é o de lhe oferecer um jeito conveniente de definir valores de opção.

A aplicação de novos valores de opção afeta apenas a janela de sessão "ativa" no momento. Se houver mais de uma janela de sessão aberta, a janela ativa é aquela que fica acima das outras janelas de sessão, com uma barra de título de uma cor diferente ou mais intensa.
