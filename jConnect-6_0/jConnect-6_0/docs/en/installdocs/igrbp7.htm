Chapter 2 Using jConnect Documentation

Programmer's reference

The jConnect for JDBC Programmer's Reference provides information and examples specific to the jConnect driver, as well as solutions and workarounds for problems you may encounter when using jConnect.

When you install jConnect, a docs directory is created under the jConnect installation directory. The docs directory contains an en/progref subdirectory containing the jConnect for JDBC Programmer's Reference in HTML and PDF formats. To obtain this reference in languages other than English, download the docsINT.zip file from the jConnect download Web site. See "Install jConnect products" for instructions.
