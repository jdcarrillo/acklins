Chapter 2 Using jConnect Documentation

jConnect documentation access using index.html files

Using a Web browser, you can access both the jConnect for JDBC Programmer's Reference and the jConnect javadoc information from an index.html file:

  • $JDBC_HOME/index.html (UNIX or Linux) and
    %JDBC_HOME%\index.html (Windows) contain links for accessing the jConnect documentation and for running sample JDBC applets.
  • $JDBC_HOME/docs/index.html (UNIX or Linux) and %JDBC_HOME%\docs\index.html (Windows) contain links to the jConnect documentation.
