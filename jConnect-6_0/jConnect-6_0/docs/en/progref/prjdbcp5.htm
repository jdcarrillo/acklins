Chapter 2 Programming Information

This chapter describes the basic components and programming requirements that comprise jConnect for JDBC. It explains how to invoke the jConnect driver, set connection properties, and connect to a database server. It also contains information about using jConnect features.

   For information about JDBC programming, go to http://java.sun.com/jdbc.To access the JDBC Guide: Getting Started manual for JDBC 1.0, go to http://java.sun.com/products/jdk/1.1/docs/guide/jdbc.To access the JDBC Guide: Getting Started manual for JDBC 2.1, go to http://java.sun.com/products/jdk/1.2/docs/guide/jdbc/.
